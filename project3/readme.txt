Compute the statistics of the highest and lowest temperature of the weather data set.
Run on a single simulated Hadoop
1. Install hadoop and jdk on your machine, the default location is /opt/module/hadoop-2.8.0.  As for HuaWei cloud machine, please use jdk-8u232-linux-aarch64.tar.gz.
2. The JAVE_HOME and HADOOP_HOME need to be set.
3. Run run.sh, jar is used for run.sh and three .java files are source code. The results are in their folder respectively.
